Web basics

HTML, CSS and vanilla JavaScript represent the core of web development. Personally, I truly believe that a good understanding of these technologies is a prerequisite to craft valuable websites. As a result, I consider it to be more important to handle properly standard JavaScript than knowing heavy frameworks.

Web Frameworks

Reinventing the wheel is often attractive but not really productive and let's be fair, it's hard to do a better job than the Google's or Facebook's dev teams. Frameworks are ruling the web nowadays and following the trends in this field is one of the best ways to understand the field.

Mobile

Mobile development is a field in which I really want to improve my skills. Making software for mobile platforms is not only about new technologies but also about understanding how customers are using their devices and what they need. It requires a totally new approach to build a well designed, useful and successful mobile application.

Back end

Back end development is a little more straightforward when it comes to languages but often more complex and accomplished as a conception drill. The mental gymnastics of complex conception is stimulating and interesting. Moreover, great conception is the required foundation of sustainable software.

Java Frameworks

Java development is framed by a few powerful and experienced frameworks. These tools serve different purposes, from the management of the lifecycle of your software to the injection of dependencies at the code level. Handling perfectly any of these frameworks asks for years of dedicated work, but a global understanding of most of them is nonetheless required to optimize technical decisions.
