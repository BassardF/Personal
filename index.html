Targeting mobile platforms

Developing software for mobile devices is one of the biggest actual challenges. Up to now, I seized every opportunity to build hybrid applications, using Apache Cordova. Crafting native IOS and Android is one of my current playgrounds and I would really like getting involved into a concrete and useful project.

Server side programming

Crafting server side programs is the best way to use traditional programming concepts, algorithms and architectural patterns. As a student and a developer, I mostly worked with languages like Java and C at optimizing lifecycles, completing testing processes and improving code quality.

Project

BuddyMap is my current IT project. The core purpose is to create a light, design and practical mobile application conceived to help you meeting your friends at the events you organize. After having organized the event and invited your friends, BuddyMap will show you their position on the map, how far they are from the meeting point as well as your own way to it. No more desperate waiting for an absentminded friend.
